I’m Data Scientist

Passionate about technology, I started programming C at 17, when I was a mechatronics technician. First programming at a lower level, using PIC and communication protocols, and later programming games in C ++.

At 18 I started Computer Engineering where I was able to participate in programming marathons and scientific works of robotics, where I used my acquired skills at the technical level and could improve them. Throughout college I migrated more and more to the area of Artificial Intelligence, where I did my final work with a multi-agent system, with agents with emotions and personality traits.

Later I entered the master's degree where I learned more about machine learning techniques, and I ended up working with clustering algorithms. I am currently working as an analyst developing AI solutions such as churn, fraud, NLP and other machine learning techniques and solutions.


Now I enjoy learning about machine learning and data visualization, and I love to find patterns and understand features that exist in our world, I'm always looking for new tools and machine learning libraries. What fascinates me most about the data is how some features can tell a lot about the world we live in.
